  • snmpwalk

    • #得到取得windows端的系统进程用户数等
snmpwalk -c public -v 1 -m ALL 192.168.30.49 .1.3.6.1.2.1.25.1  

#取得系统总内存
snmpwalk -c public -v 1 -m ALL 192.168.30.49 .1.3.6.1.2.1.25.2.2  

#取得IP信息
snmpwalk -c public -v 1 -m ALL 192.168.30.49 hrSystemNumUsers

snmpwalk -c public -v 1 -m ALL 192.168.30.49 .1.3.6.1.2.1.4.20    

#查看系统信息
snmpwalk -v 2c -c public 192.168.30.49 system   

snmpwalk -v 1 192.168.30.49 -c public ifDescr

如果你也不知道具体参数，那么就执行snmpwalk -v 2c -c public 192.168.30.49，就可以看到所有信息。
